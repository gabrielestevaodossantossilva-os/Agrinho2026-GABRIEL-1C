# 🌱 Agro Forte, Futuro Sustentável

## Concurso Agrinho 2026

### Tema

**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

---

## Objetivo

Este projeto foi desenvolvido para conscientizar a população sobre a importância do equilíbrio entre a produção agrícola e a preservação do meio ambiente. O site apresenta informações sobre agricultura sustentável, os benefícios da tecnologia no campo e incentiva atitudes responsáveis para garantir um futuro melhor.

---

## Justificativa

O agronegócio é fundamental para a economia e para a produção de alimentos. Ao mesmo tempo, preservar a natureza é essencial para manter a qualidade de vida das futuras gerações. Este projeto demonstra que é possível produzir de forma sustentável utilizando tecnologias modernas, preservando rios, florestas e a biodiversidade.
